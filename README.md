# 🚀 NexusHub - Enterprise Workspace & Project Management System

NexusHub is a full-stack Enterprise Workspace and Project Management System developed to simplify employee management, project tracking, task management, attendance, leave management, and report generation in a single platform.

It provides secure JWT-based authentication, role-based access, and an interactive dashboard for managing organizational activities efficiently.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/arbitha34/nexushub.git
```

## Backend

```bash
mvn spring-boot:run
```

Backend runs at:

```
http://localhost:8080
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend runs at:

```
http://localhost:5173
```
